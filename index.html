Recent Project

Mashr

Mashr simplifies the process of taking data from disparate sources and putting it into a single database so it can be used. Mashr is optimized for data pipeline best practices including monitoring, archiving, and backup in case of failover.

Mashr is built with Node.js, and provides an easy-to-use CLI; uses Docker to host Embulk on Google Cloud Platform GCE Instances, stores data files in Google Cloud Storage, moves data with Cloud Functions, and loads data into the BigQuery database.
